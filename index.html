Draw on the field. Each pixel you paint is stamped with the carrier's local state — a 5×5 neighborhood snapshot at that point. Every mark has its own key.

Marks blur in and out. Each frame, each pixel's stored neighborhood is compared to the current carrier. Strong match = sharp, bright. Partial match = blurred, spread into neighbors, dimmed. Anti-phase = invisible. The blur is spatial — the pixel literally spreads — because holographic reconstruction degrades gracefully, not categorically.

Pause freezes the carrier for careful drawing. Different modes produce different keys. Marks drawn under one mode configuration blur out when you switch, sharpen when you switch back.

Slides let you snapshot the whole board — all marks and their carrier keys — and reload them later. Save a drawing, clear, draw something new, save again. Load any slide to restore that state into the living field. The holographic blur still applies.
